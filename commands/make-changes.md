# Make Changes

## Make changes to your repo.

### **1. Adding files to the staging**

*   #### `git add <files>`

    \-adds specified files to the staging area.\

*   **`git add .`**

    \-adds all files to the staging area.\


### 2. Committing

*   #### `git commit -m "Commit-Message"`

    \-Committing staged file to git with a message.
